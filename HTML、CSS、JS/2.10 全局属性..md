# 全局属性

## 命名属性

CSS 如何找到标签呢？可以直接用标签名，也可以使用 `class` 属性给标签命名，还有 `id` 属性。





## `style`

可以给标签提交行内样式。





## `title`

鼠标在添加了 `title` 属性的标签上停留则会出现提示。





## `contenteditable`

让此标签内容可被修改。





## `tabindex`

网页摁 `tab` 切换到对应内容。

从高到低的顺序切换，但是不会切到为负数的标签。





## `data-*`

`data-nickname` 添加自定义内容给标签。





## 问题

### `class` 和 `id` 有什么区别？

特别大的区块 可以加 `id` 其他都 `class` 。



### Web页面分为哪三层，分别是什么？

结构层、表现层、行为层



### 聊一聊HTML的语义化？

便于开发者阅读；页面如果没有语义化CSS的选择会很不方便；方便机器和搜索引擎。
